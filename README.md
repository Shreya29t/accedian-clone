# Internship Assignment - Full Stack Project

## Description
This project was developed as part of an internship assignment. It is a web application built using Next.js that demonstrates core frontend and basic full stack development concepts such as component-based architecture and responsive design.

## Tech Stack
- Next.js
- React
- JavaScript / TypeScript
- CSS / Tailwind CSS

## Features
- Responsive user interface
- Reusable components
- Clean and structured code
- Fast performance using Next.js

## How to Run
1. Clone the repository
2. Install dependencies using:
   npm install
3. Start the development server:
   npm run dev
4. Open http://localhost:3000 in your browser

## Project Structure
- app/ → Main application pages
- components/ → Reusable UI components
- public/ → Static files
